# Character-Level Transformer Language Model

This project implements a Transformer-based language model from scratch at the character level.

## Features
- Custom tokenizer (character-level)
- Multi-head self-attention implementation
- Transformer blocks with feed-forward layers
- Training pipeline with cross-entropy loss
- Evaluation using perplexity and bits-per-character

## Status
🚧 Ongoing (Stanford CS336-inspired implementation)

## Goals
- Train a 6–12 layer transformer on text corpus
- Analyze long-range dependencies
- Perform ablation studies on architecture components

## Tech Stack

- PyTorch
